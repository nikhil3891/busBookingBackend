
# 🚌 Bus Booking Backend (Node.js + TypeScript + MongoDB)

A production-quality backend built using **Node.js**, **TypeScript**, and **MongoDB** for managing users, admins, buses, ticket bookings, and email notifications.

---

## 🧱 Tech Stack

| Layer | Technology Used |
|-------|------------------|
| Backend Runtime | Node.js (TypeScript) |
| Framework | Express.js |
| Database | MongoDB (via Mongoose) |
| Authentication | Passport.js + JWT |
| Validation | Joi |
| Mailing | Nodemailer + EJS templates |
| Security | Helmet, Rate Limiter, CORS |
| Logger | Morgan |
| Package Manager | pnpm |
| Environment | dotenv |
| Testing | Postman / Jest (optional) |
